## 🕹 How to Play

- Click one of the three buttons: **Rock**, **Paper**, or **Scissors**.  
- The computer will randomly pick a move.  
- Your score updates automatically: **Wins**, **Losses**, **Ties**.  
- Toggle background music using the 🔊 button.  
- Reset score using the **Reset Score** button.
